# School Logo Files

Place your school logo files here:

- `logo.svg` - Main logo in SVG format (preferred)
- `logo.png` - Main logo in PNG format (fallback)
- `logo-icon.svg` - Square icon version (48x48px)
- `logo-icon.png` - Square icon version (48x48px)

The navigation will automatically use the SVG logo, and fall back to PNG if SVG is not available.
